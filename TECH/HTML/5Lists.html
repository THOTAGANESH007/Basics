Semantic HTML (DL 2)

Definition: Elements that clearly describe their content's purpose and relationship to the web page.
Examples: header, nav, article, footer, main, section, h1, form, table,p


Non Semantic HTML

Definition: Elements that don't convey any specific meaning about the content they enclose.
Examples: div (a generic block-level container) and span (a generic inline container).
