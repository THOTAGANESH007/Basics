Transition:
The transition is triggered when there is change in the element's properties (:hover, :active, :focus, :checked)
Transform(2D):
translate(),rotate(),scaleX(),scaleY(),scale(),skewX(),skewY(),skew(),matrix()
Transform(3D):
rotateX(), rotateY(), rotateZ()
  • transition-property (name)
  • transition-duration (in seconds)
  • transition-timing-function:
    • ease: start slow, go fast, end slow (default)
    • ease-in: start slow
    • ease-out: end slow
    • ease-in-out: slow start and slow end
    • linear: same speed from start to end
    • cubic-bezier(n,n,n,n)
  • transition-delay: (in seconds)
