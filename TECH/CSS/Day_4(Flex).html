A Flexible Layout must have a parent element with the display property set to flex.

Direct child elements(s) of the flexible container automatically becomes flexible items.

flex-direction:

  1. row(default)
  2. column
  3. row-reverse
  4. column-reverse

flex-wrap

  1. no-wrap(default)
  2. wrap
  3. wrap-reverse

The flex-flow property is a shorthand property for the flex-direction and the flex-wrap properties. (flex-flow: _ _)

justify-content

  1. center: aligns the flex items at the center of the container:
  2. flex-start: the flex items at the beginning of the container (default)
  3. flex-end: the flex items are at the end of the document
  4. space-around: displays the flex items with space before, between, and after the lines
  5. space-between: displays the flex items with space between the lines
  6. space-evenly: displays the flex items with equal space around them
align-items && align-content

center flex-start flex-end stretch baseline normal

Flex Items: The direct children of the flex container

Order: It specifies the order of the items in the flex container(numbers)(0)

flex-grow:num,how much a flex item should grow relative to the other items

flex-shrink: how much a flex item will shrink relative to the other items(1)

flex-basis:px initial length of the flex item

flex: shorthand for the flex-grow, flex-shrink, flex-basis
